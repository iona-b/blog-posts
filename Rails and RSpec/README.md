# RSpec and Rails

## Installing Rails RSpec

https://rubygems.org/gems/rspec-rails/versions/3.4.2

1. gem 'rspec-rails', '~> 3.4', '>= 3.4.2' in group :development, :test do
2. bundle install
3. rails generate rspec:install
